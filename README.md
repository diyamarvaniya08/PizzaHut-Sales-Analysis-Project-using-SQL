# PizzaHut-Sales-Analysis-Project-using-SQL
SQL-based analysis of PizzaHut sales data to derive meaningful business insights.

# Project Overview
 - This project focuses on analyzing PizzaHut sales data using SQL to derive meaningful business insights.
 - The analysis answers real-world business questions related to revenue, sales trends, customer ordering behavior, and product performance.

# Dataset Description & tool used
 - The dataset contains PizzaHut sales records and includes the following tables:
     - orders
     - order_details
     - pizzas
     - pizza_types
 - The data includes order details, pizza categories, sizes, prices, and quantities sold.
- Tool used:
     - MySQL
       
# What I worked on
 - SQL (JOINs, GROUP BY, Aggregations)
 - Where, Having Clause
 - Subqueries, Window Functions
 - Business Problem Solving
   
# Key Analysis Performed
 - Revenue analysis using aggregate functions
 - Top-selling pizzas and categories
 - Order quantity and sales trend analysis
 - Use of JOINs, GROUP BY, and subqueries
